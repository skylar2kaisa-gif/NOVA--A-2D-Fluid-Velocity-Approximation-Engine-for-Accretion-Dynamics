# NOVA--A-2D-Fluid-Velocity-Approximation-Engine-for-Accretion-Dynamics
This script provides a lightweight, flicker-free terminal simulator that models stellar bodies trapped in an energy-dissipating, viscous space-time metric. Instead of solving standard vacuum geodesic tensors, the engine utilizes a direct radial-velocity decay model ((\frac{dr}{dt} \propto -r^{-2})) coupled with a uniform frame-dragging angular velocity ((\frac{d\theta}{dt} = \Omega + \delta)). This efficiently yields a realistic differential orbital decay gradient without the high computational overhead of full numerical relativity integration.
